how to Run
Requirements:

Download and Install local web server such as XAMPP
start your local web server (Xampp) amd start Apache and Mysql
Installation

Extract the source code zip file.
If you are using XAMPP, copy the extracted source code folder and paste it into the XAMPP's "htdocs" directory. 
Locate the provide SQL file in the extracted source code folder. The file is known as "db_shop".
Open a browser and browse the PHPMyAdmin. i.e. (http://localhost/phpmyadmin)
Create a new database naming "db_shop".
Import the SQL file in your newly created database.
Open a new window/tab in your browser and browse the web application. 
to access the admin panel type this in the web browser. i.e.(http://localhost/sanga/admin) 
for the website/ customer panel type this in the web browser (http://localhost/sanga/)
